# ポモドーロタイマー
[アプリページ](https://ryochinbo.github.io/timer_Pomodoro/)
## 概要
このプロジェクトは、ポモドーロ・テクニックをサポートするために設計されたシンプルなWebベースのタイマーアプリケーションです。集中して作業を行い、適切な休憩を取るサイクルを助けます。


## 使い方

1.  **ブラウザで開く**: `index.html`ファイルをWebブラウザで開きます。
2.  **時間設定**:
    *   「実施時間」と「休憩時間」のドロップダウンメニューからプリセットの時間を選択します。
    *   ドロップダウンの隣にある入力フィールドに直接数字を入力することで、任意のカスタム時間を設定することも可能です。
3.  **タイマー操作**:
    *   「スタート」ボタンでタイマーを開始します。
    *   「ストップ」ボタンでタイマーを一時停止します。
    *   「リセット」ボタンでタイマーを初期状態に戻します。

## セットアップ

### ローカルでの実行

1.  このリポジトリをクローンします。
    ```bash
    git clone https://github.com/YOUR_USERNAME/timer_Pomodoro.git
    ```
    (YOUR_USERNAMEとリポジトリ名をあなたのものに置き換えてください)
2.  クローンしたディレクトリに移動します。
    ```bash
    cd timer_Pomodoro
    ```
3.  `index.html`ファイルをWebブラウザで開きます。

### 音声ファイルについて
セッション切り替え時の通知音と休憩終了前のカウントダウン音を有効にするには、以下のファイルをプロジェクトのルートディレクトリに配置する必要があります。
*   `notification.mp3` (セッション切り替わり時の通知音)
*   `countdown.mp3` (休憩終了10秒前のカウントダウン音)
これらのファイルがない場合でもアプリケーションは動作しますが、音声通知は鳴りません。
